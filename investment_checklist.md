# 投資チェックリスト（ファンダメンタル分析）

---

## 1. 損益計算書(P/L)

| 視点 | チェック項目 | 基準 |
|------|------------|------|
| 成長性 | 売上高成長率(YoY) | +10%以上で合格、+20%以上で優秀、+30%以上でトップクラス |
| 成長性 | 増収増益かどうか | 増収減益は不合格。利益率が上がっているか確認 |
| 収益性 | 営業利益率 | 小売・卸売1〜5% / 製造5〜10% / 情報通信15% / 医薬品30%超 |
| 収益性 | 販管費の中身 | 研究開発・広告宣伝費は良い。役員報酬・交際費・その他は要注意 |
| 将来性 | 進捗率(2Q時点) | 55%以上で良い、45%以下で要注意。季節性に注意 |
| 将来性 | MR&Aのコメント | 進捗率を裏付ける具体的な根拠があるか |

## 2. 貸借対照表(B/S)

| 視点 | 指標 | 計算式 | 基準 |
|------|------|--------|------|
| 安全性 | 自己資本比率 | 自己資本 / 総資産 × 100 | 40%以上で安心、50%以上で鉄壁 ※銀行・インフラ・急成長企業は低くてもOK |
| 支払能力 | 流動比率 | 流動資産 / 流動負債 × 100 | 100%未満で危険、150%以上で安心、200%以上で非常に安全 |
| 支払能力 | 当座比率 | 当座資産 / 流動負債 × 100 | 70%未満で注意、70%以上で問題なし、100%以上で超安全 |
| 借金の質 | D/Eレシオ | 有利子負債 / 自己資本 | 0.5倍以下で非常に安全、1.0倍以下で健全、2倍以上で要注意 ※利益が大きく伸びていればOK |

---

## 3. キャッシュフロー計算書(C/F)

| 視点 | チェック項目 | 基準 |
|------|------------|------|
| 利益の質 | 営業CF vs 営業利益 | 営業CF > 営業利益なら質が高い。下回る場合は売掛金・棚卸資産の増加を確認 |
| 利益の質 | 営業CFマージン | 営業CFマージン(営業CF / 売上高)が営業利益率より高ければ優良 |
| 投資姿勢 | 投資CF | マイナスが正常。プラスの場合は資産売却の可能性があり要注意 |
| 財務健全性 | FCF | FCF = 営業CF − 投資CF。プラスであれば自由に使えるキャッシュがある |

---

## 4. 定性分析

| チェック項目 | 観点 |
|------------|------|
| コメントの具体性 | 抽象的な表現でなく、数字や具体策が書かれているか |
| 未来への戦略 | 成長のロードマップが明確か |
| 過去からの一貫性 | 以前の方針と矛盾していないか |
| リスクの把握 | 事業リスクを自覚し、対策が示されているか |
| 主語が自分か | 「市場が〜」ではなく「当社が〜」という表現か |
| 株主への姿勢 | 株主をパートナーとして捉えているか |

---

## 5. バリュエーション

| 指標 | 計算式 | 目安 |
|------|--------|------|
| PER | 株価 / EPS | 15倍程度が標準。高成長企業は割高でも許容される場合あり |
| PBR | 株価 / BPS | 1倍以下は解散価値割れ。他指標が良ければ割安のチャンスかも |
| PEGレシオ | PER / EPS成長率 | 1.0未満なら成長性に対して割安の可能性 |
